# 깊은샘 묵상 iPhone PWA 사용 안내

이 문서는 GitHub Pages에 배포된 `깊은샘 묵상` iPhone용 PWA를 설치하고, 성경 JSON 데이터팩을 추가하는 방법입니다.

## 1. GitHub Pages 주소 열기

GitHub Pages 배포 주소:

```text
https://saintfoolk.github.io/iphone-bible-meditation-pwa/
```

iPhone에서는 반드시 Safari로 위 주소를 여세요. Chrome이나 카카오톡 내부 브라우저에서는 홈 화면 앱 설치가 제한될 수 있습니다.

## 2. 홈 화면에 앱으로 추가하기

1. iPhone Safari에서 위 주소를 엽니다.
2. 첫 화면이 보이면 하단의 공유 버튼을 누릅니다.
3. 메뉴에서 `홈 화면에 추가`를 선택합니다.
4. 이름을 `깊은샘 묵상`으로 둡니다.
5. 우측 상단의 `추가`를 누릅니다.
6. 홈 화면에 생긴 `깊은샘 묵상` 아이콘으로 앱을 실행합니다.

처음 접속할 때는 인터넷 연결이 필요합니다. 한 번 열린 뒤에는 앱 화면과 기본 데이터가 브라우저 캐시에 저장됩니다.

## 3. 기본으로 포함된 성경

PWA 기본 배포본에는 용량과 배포 안정성을 위해 두 번역본만 포함되어 있습니다.

- 개역한글 `korhrv`
- WEB `engwebp`

다른 성경은 앱의 `자료` 화면에서 JSON 데이터팩으로 추가합니다.

## 4. 추가 성경 JSON 준비하기

추가 성경은 각각 하나의 `.json` 파일입니다. PC에 생성된 추가 데이터팩 위치는 다음과 같습니다.

```text
C:\Users\gram\Documents\성경 묵상앱\dist\pwa-extra-bible-data\translations
```

예시:

- `korkct2024.json`: 공동번역 개정판
- `kornkt2024.json`: 새한글성경
- `korwrm.json`: 우리말성경
- `kor200nt.json`: 200주년 기념 신약
- `koreasy.json`: 쉬운말 성경

이 JSON 파일을 iPhone의 `파일` 앱에서 접근할 수 있는 위치로 옮깁니다.

가능한 방법:

- iCloud Drive에 업로드
- AirDrop으로 iPhone에 전송
- Google Drive, Dropbox 등에 올린 뒤 파일 앱에서 열기
- 카카오톡이나 메일로 받은 뒤 파일 앱에 저장

## 5. 앱 안에서 성경 추가하기

1. 홈 화면의 `깊은샘 묵상` 앱을 엽니다.
2. 좌측 상단의 책 아이콘을 눌러 `자료` 화면으로 갑니다.
3. `데이터팩 성경` 영역을 찾습니다.
4. `데이터팩 폴더 선택`을 누릅니다.
5. iPhone 파일 앱에서 추가할 `.json` 파일 또는 JSON 파일들이 들어 있는 폴더를 선택합니다.
6. 앱에 번역본 목록이 나타나면 추가할 성경을 체크합니다.
7. `선택한 성경 저장`을 누릅니다.
8. 저장이 끝나면 `읽기`, `검색`, `성서일과`, `묵상` 화면에서 해당 번역본을 선택할 수 있습니다.

만약 iPhone에서 폴더 선택이 잘 되지 않으면 JSON 파일을 하나씩 선택해서 추가하세요.

## 6. 성경 추가 후 확인하기

### 읽기 화면

1. `읽기` 탭으로 이동합니다.
2. 본문 선택 버튼을 누릅니다.
3. `번역본` 목록에서 추가한 성경을 선택합니다.
4. 성경, 장, 절을 고르고 적용합니다.

### 검색 화면

1. `검색` 탭으로 이동합니다.
2. 검색어를 입력합니다.
3. 번역본을 선택하거나 전체 번역본으로 둡니다.
4. 검색 결과에서 본문으로 이동합니다.

### 성서일과/묵상 화면

1. `성서일과` 또는 `묵상` 화면으로 이동합니다.
2. 설정을 열고 번역본을 선택합니다.
3. 선택한 번역본은 변경하기 전까지 유지됩니다.

## 7. 문제가 생길 때

### 앱이 빈 화면으로 열릴 때

- Safari에서 주소가 정확한지 확인합니다.
- 주소는 반드시 아래처럼 저장소 이름까지 포함해야 합니다.

```text
https://saintfoolk.github.io/iphone-bible-meditation-pwa/
```

### 오래된 화면이 계속 보일 때

1. iPhone 설정 앱을 엽니다.
2. Safari로 이동합니다.
3. 방문 기록 및 웹 사이트 데이터 지우기를 실행합니다.
4. 다시 Safari에서 Pages 주소를 열고 홈 화면에 추가합니다.

### 성경 JSON이 보이지 않을 때

- 파일 확장자가 `.json`인지 확인합니다.
- 압축 파일 상태라면 먼저 압축을 풉니다.
- 파일 앱에서 접근 가능한 위치에 저장했는지 확인합니다.

## 8. 권장 설치 순서

1. GitHub Pages 주소를 Safari에서 열기
2. 홈 화면에 추가
3. 홈 화면 아이콘으로 앱 실행
4. 읽기 화면에서 개역한글과 WEB 확인
5. 추가 성경 JSON을 iPhone 파일 앱에 저장
6. 자료 화면에서 JSON 성경 추가
7. 읽기, 검색, 성서일과, 묵상에서 번역본 선택 확인
